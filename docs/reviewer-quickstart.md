# Reviewer Quickstart

Use this path to understand the repository's purpose, representative controls, and evidence model in 10-15 minutes.

## What This Repo Is

This repository is an artifact-first Agentic SDLC control-plane lab. It focuses on GitHub-native workflows, schemas, hooks, MCP examples, logs, memory, and governance artifacts. It is built for practical AI agent infrastructure literacy and repo-based certification cram support.

## What This Repo Is Not

- Not a production security platform.
- Not an exploit lab.
- Not an official certification guide.
- Not a claim of exam coverage or exam success.
- Not a live agent execution system.

## 15-Minute Review Path

1. Read `README.md`.
2. Open `docs/vulnerability-to-defense-matrix.md`.
3. Open `labs/README.md`.
4. Review Labs 01, 05, 09, and 11.
5. Inspect `artifacts/` and `schemas/`.
6. Run `python scripts/validate_repo.py`.
