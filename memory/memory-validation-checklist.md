# Memory Validation Checklist

Use this checklist before relying on stored memory.

- Does the memory have a source artifact?
- Does the source artifact still exist?
- Has the relevant file changed since the memory was recorded?
- Is the memory scoped to the current repository, user, branch, or task?
- Is the memory expired?
- Does it conflict with current repository files?
- Does it increase permission or autonomy beyond current policy?
- Should it be pruned, reset, or revalidated?

## Rule

Current repository evidence outranks stale memory.
