# Agent Compliance Review

## Review Questions

1. Can the agent mutate repository state?
2. Can the agent execute commands?
3. Can the agent access remote MCP servers?
4. Can the agent read sensitive files?
5. Can the agent change workflows or permissions?
6. Are outputs preserved as artifacts?
7. Is there a human approval path for high-risk actions?
8. Can a reviewer reconstruct what happened from logs and artifacts?
